# Gainwell Genie Data App (APX)

Databricks App: ask questions in natural language, view data, download CSV, and visualize.

- **Backend**: FastAPI + Genie (NL-to-SQL)
- **Frontend**: React + TanStack Router + shadcn/ui (APX)
- **Run**: From repo root see main README; from here: `uv run apx dev start` or `uv run apx build`
